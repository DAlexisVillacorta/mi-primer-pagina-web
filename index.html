¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo que se usa para controlar la apariencia de las páginas web. Permite definir colores, tipografías, márgenes, posiciones y efectos visuales.

Conceptos clave de CSS

  • Seletores: Identifican qué elementos serán estilizados.
  • Propiedades: Definen qué características visuales cambiar.
  • Valores: Especifican el resultado del estilo (ej. `color: blue`).
  • Box Model: Concepto clave sobre el tamaño y espaciado de los elementos.
  • Flexbox y Grid: Métodos modernos para diseñar estructuras responsivas.

Ejemplo de código CSS

Aquí tienes un ejemplo de cómo cambiar el color de fondo y el color del texto en CSS:


body {
    background-color: black;
    color: white;
    font-family: Arial, sans-serif;
}

También podemos estilizar botones con efectos de neón:


button {
    background-color: black;
    color: cyan;
    border: 2px solid cyan;
    padding: 10px 20px;
    border-radius: 5px;
    box-shadow: 0 0 10px cyan;
}
